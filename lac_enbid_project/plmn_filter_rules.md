# 网优 cell 项目：PLMN 过滤规则（第一步）

目标：在对表 `网优cell项目_清洗补齐库_v1` 做清洗时，**第一步尽量过滤无效 / 非目标 PLMN**，只保留中国境内的主要运营商及铁路专网，用作后续分析和项目匹配的基础数据集。

## 1. 保留范围（有效 PLMN）

> 规则：**第一步只使用上述 12 个 PLMN，对应移动 / 联通 / 电信 / 广电 / 铁路专网。**

## 2. 排除范围（本步不使用的 PLMN）

以下情况在第一步统一视为“无效或非目标”，全部过滤掉，不参与后续计算：

- 任意 `运营商id` 不在上述 12 个 PLMN 列表中；
- `运营商id` 为 NULL；
- 明显异常或疑似脏数据的编码（例如 `"(null)(null)"`、`460460` 等）；
- 国外 MCC/MNC（如 `510xx`、`520xx`、`404xx`、`452xx`、`470xx`、`286xx` 等）。

> 这些数据可在后续单独分析（如检查数据来源、质量、设备漫游情况等），但**不参与第一步项目识别与统计**。
