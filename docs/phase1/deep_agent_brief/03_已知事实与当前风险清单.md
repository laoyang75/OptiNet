## 2. 已知关键风险（请作为方案输入）
1. 指标聚合链路存在口径不一致风险。
- 现象：指标汇总表和事实表有差值。
- 影响：自动门禁可能误报，影响决策。

2. Layer_5 关键异常字段缺失风险。
- 现象：部分异常字段在明细层有，但画像层未显式暴露。
- 影响：下游只看画像时可能看不到关键风险。

3. Layer_1 入口口径存在“文档对象 vs 实际对象”不一致。
- 影响：接手人员会混淆标准入口，排障路径不唯一。

4. 文档之间存在口径冲突。
- 现象：同一策略在不同文档表述冲突。
- 影响：执行与验收预期不一致。
