## 4. 最终冻结规则

1. **唯一真相包**：`03_final/00_05` 六文件共同构成正式冻结包。
2. **唯一数据库事实来源**：计数、比例、差集、缺口统一以 `PG17 execute_sql` 为准；结构确认统一以 `PG17 search_objects` 为准。
3. **唯一页面证据来源**：只允许承接 round3 merged 结论与已归档截图，不重新跑页面、不新增页面心证。
4. **唯一主链来源**：rebuild4 正式主链只接受 `rebuild2.l0_gps + rebuild2.l0_lac` 重跑生成的 real 数据；`rebuild3*` 永远只读参考。
5. **唯一裁决集**：只承认 round1 `D-001 ~ D-003` 与 round2 / round3 “无新增裁决”的结论。
6. **唯一上抛条件**：只有规则 seed 不可用、real rolling 窗口缺失、页面一级主语冲突这三类异常真正发生时，才允许新增裁决文件。
