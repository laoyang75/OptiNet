# 目录速查

## 人类常看目录

- Runbook：`rebuild4/docs/00_runbook/`
- 当前轮合并稿：`rebuild4/docs/02_rounds/<round>/merged/`
- 当前轮裁决文件：`rebuild4/docs/02_rounds/<round>/decisions/`
- 最终冻结稿：`rebuild4/docs/03_final/`

## Agent 常看目录

- 输入资料：`rebuild4/docs/01_inputs/`
- 当前轮 prompt：`rebuild4/docs/02_rounds/<round>/prompts/`
- 当前轮输出目录：`rebuild4/docs/02_rounds/<round>/outputs/`
