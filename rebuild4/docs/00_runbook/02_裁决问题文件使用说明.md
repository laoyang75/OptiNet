# 裁决问题文件使用说明

每轮如果存在真正需要你决定的问题，会生成：

- `rebuild4/docs/02_rounds/<round>/decisions/01_待裁决问题清单.md`

## 1. 每个问题的固定结构

每个问题必须包含：

1. 问题编号
2. 问题标题
3. 为什么出现分歧
4. 3 个 agent 的不同结论
5. 我作为主控的解释
6. A / B / C 三个选项
7. 每个选项的影响
8. 我的推荐选项
9. 你的回复格式

## 2. 你的最小工作量

你只需要回复编号和选项，例如：

- `D-001: A`
- `D-003: B`

## 3. 什么叫好的选项设计

A / B / C 必须满足：

- 真的是不同策略，而不是换个说法
- 每个选项都要告诉你影响
- 我必须给出推荐项
- 不允许把模糊选项丢给你做二次设计
