# 画像管道 — 总览

## 这是什么

画像是 rebuild4 的第二步处理管道。ETL 把原始数据变成了结构化的 `etl_filled`，画像则从中提炼出每个实体（Cell / BS / LAC）的特征参数。

这些参数是流转系统做判断的基础：一个 Cell 是否搬迁了、一个 BS 覆盖是否异常、一个 LAC 的数据质量是否可信——都要从画像参数中读取。

## 三层实体

```
LAC (位置区)
 └── BS (基站)
      └── Cell (小区)  ← 最小分析单元
```

**自下而上构建**：先算 Cell，再从 Cell 聚合出 BS，再从 BS 聚合出 LAC。

## 为什么没有初始化

流式评估实验 (详见 `06_流式评估.md`) 证明：逐天累积的流式画像与全量批量计算**数学等价** (Day 7 质心偏差 = 0.00m，生命周期一致率 = 98.9%)。

因此删除了 11 步初始化流程，改为 `profile.py` 直接全量构建。画像管道可随 ETL 一起运行，也可独立触发。

```
etl_filled (688K 条, 66 列)
    │
    ▼
┌─────────────────────────────┐
│ Step 1: 独立观测点            │
│                               │
│ 去重 key: (cell_id, minute)  │
│ GPS 仅用原始有效值 (中国边界) │
│ 信号仅用合理范围              │
│                               │
│ 产出: 158K 条独立观测         │
└──────────────┬──────────────┘
               │
               ▼

│                               │
│ 质心: PERCENTILE_CONT(0.5)   │
│       (中位数, 不用 AVG)      │
│ P50/P90: 距质心距离百分位     │
│ BS 质心 → Cell→BS 距离        │
│                               │
│ 产出: 20K Cell 质心           │
└──────────────┬──────────────┘
               │
               ▼

│                               │
│ 每日质心 → max_spread + ratio │
│ 分类: stable / collision /    │

│                               │

└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│ Step 4: 组装 Cell 画像        │
│                               │
│ position_grade (4 级)         │

│            active             │

│                               │

└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│ Step 5: BS 画像               │
│ 从 Cell 聚合 → etl_dim_bs    │

               │
               ▼

│ 从 BS 聚合 → etl_dim_lac     │
│ 产出: 1,270 条               │
└─────────────────────────────┘
```

## 数据表

| 表 | Schema | 说明 |
|----|--------|------|
| `etl_filled` | rebuild4 | 输入: ETL 补齐后的最终产出 (688K 条, 66 列) |
| `etl_dim_cell` | rebuild4 | Cell 画像 (20,394 条, 42 列) |
| `etl_dim_bs` | rebuild4 | BS 画像 (11,942 条, 38 列) |
| `etl_dim_lac` | rebuild4 | LAC 画像 (1,270 条) |
| `eval_stream_convergence` | rebuild4 | 流式评估收敛曲线 (7 行) |

## 与 rebuild2 的对照

| rebuild2 | rebuild4 | 差异 |
|----------|----------|------|
| `dim_cell_refined` + `_sample_cell_profile_summary` | `etl_dim_cell` | 质心用中位数替代 AVG，偏差 19.3m |
| `dim_bs_refined` + `_sample_bs_profile_summary` | `etl_dim_bs` | 从 Cell 聚合，已完成 |
| `dim_lac_trusted` | `etl_dim_lac` | 从 BS 聚合，已完成 |
