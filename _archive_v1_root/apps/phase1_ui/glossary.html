基础对象与分层

L0~L5

分层处理链路：L0 原始数据，L2 清洗聚合，L3 基站层，L4 修复补齐层，L5 画像输出层。

BS (Base Station)

基站对象。一个基站下面可以挂多个小区（CELL）。

CELL

小区对象，是无线网络里更细粒度的覆盖单元。

LAC (Location Area Code)

位置区码，用于核心网位置归属和区域管理。

EN 视图

英文列名视图（English Naming），便于对接外部系统。

Trace / seq_id

样本穿透追踪；`seq_id` 是单条采样记录的稳定主键。

信号指标与单位

RSRP

参考信号接收功率，反映覆盖强弱，数值越高（越接近 0）通常越好。

RSRQ

参考信号接收质量，反映信号质量和干扰水平。

SINR

信号与干扰加噪声比，反映链路质量，越高通常越好。

RSSI

接收信号总强度，包含有用信号与干扰。

dBm / ASU

dBm 是绝对功率单位；ASU 是终端侧离散强度等级。

P50 / P90

分位数指标：P50 为中位值，P90 为高位阈值。

流程步骤与门禁术语

Step40

位置补齐步骤，处理 GPS 缺失、异常与中心点回填。

Step41

信号字段补齐步骤，修复 RSRP/RSRQ/SINR 等缺失。

Step43

指标汇总步骤，负责把过程指标归集到汇总表用于对账。

Step52

CELL 画像步骤，基于过滤口径产出小区级画像。

Not_Filled

未补齐项，表示当前记录在对应步骤中仍未被修复。

Severe Fill

严重碰撞场景的补齐统计项，用于高风险补齐监控。

bs_id_lt_256

基站 ID 小于 256 的异常标签，常用于识别疑似异常编码。

PASS / FAIL

门禁校验通过/未通过状态。
