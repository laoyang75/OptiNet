# ENBID 研究用：5G `cell_id` TOP 20（已剔除已知异常）

> 说明：本文件给出在 **已筛选的 L1-CELL 合规明细** 上统计的  
> 5G `cell_id` 访问量 TOP 20，已剔除已知无效/溢出值 `2147483647`。  
> 后续 ENBID 研究的第一步，就以这 20 个 `cell_id` 为切入点。

统计基于视图：

- `public.v_cell_L1_stage1`  
  - 定义：见 `Layer_1/Cell/Cell_Filter_Rules_v1.md`，核心过滤包括：  
    - 5 个核心 PLMN；  
    - `原始lac` 纯数字；  
    - 只保留 4G/5G；  
    - `cell_id` 非空、纯数字、>0；  
    - 剔除 `cell_id = 2147483647`（溢出默认值）。

---

## 1. 全网 5G `cell_id` TOP 20（按记录数排序）

统计 SQL（供复现参考）：

```sql
SELECT
  t.cell_id::bigint AS cell_dec,
  COUNT(*) AS cnt
FROM public.v_cell_L1_stage1 t
WHERE lower(t.tech) = '5g'
GROUP BY cell_dec
ORDER BY cnt DESC
LIMIT 20;
```
