## 0) 字段/缩写速查（看表用）

- `PASS/WARN/FAIL`：通过/告警/阻断
- `OK（未触发阈值）`：未触发该项的默认阈值告警（等价于“正常”）
- `TopN/Top10`：前 N / 前 10（用于抽样展示）
- `operator_id_raw`：运营商 ID（MCC/MNC，示例：46000/46001/46011/46015）
- `tech_norm`：制式（归一化技术体制，示例：4G/5G）
- `bs_id`：基站 ID
- `cell_id_dec`：小区 ID（十进制）
- `gps_status`：原始 GPS 状态（Verified=可信，Missing=缺失，Drift=漂移）
- `gps_status_final`：修正/补齐后的 GPS 状态（Verified=最终有GPS，Missing=最终无GPS）
- `gps_source`：回填/纠偏来源
  - `Original_Verified`：原始可信（不改）
  - `Augmented_from_BS`：用 Usable 基站中心点回填/纠偏
  - `Augmented_from_Risk_BS`：用 Risk 基站中心点回填/纠偏（风险来源，需显式标记）
  - `Not_Filled`：无法回填
- `is_from_risk_bs`：是否来自 Risk 基站（1=是，0=否）
- `gps_dist_to_bs_m`：原始 GPS 点到“最终基站中心点”的距离（单位：米）
- `lon_raw/lat_raw`：原始经纬度
- `lon_final/lat_final`：最终经纬度（修正/回填后）
- `report_date`：统计日期（日粒度）
- `row_cnt`：行数
