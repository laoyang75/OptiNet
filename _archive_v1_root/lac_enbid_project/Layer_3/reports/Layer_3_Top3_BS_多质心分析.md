# Layer_3 Top3 疑似碰撞桶：多质心分析（按 `cell_问题分析.md` 方法复现）

目的：对 `lac_enbid_project/Layer_3/reports/Layer_3_Collision_Suspects_With_Cells.md` 中**前 3 个桶**，按 `lac_enbid_project/Layer_3/notes/cell_问题分析.md` 的“去极值→聚焦 20–50km→四分位聚类→时间重叠判断”方法做同款分析，判断是否属于同类问题。

数据来源：

- BS 桶中心/碰撞标记：`public."Y_codex_Layer3_Step30_Master_BS_Library"`

- Step05 多 LAC 异常：`public."Y_codex_Layer2_Step05_Anomaly_Cell_Multi_Lac"`

说明（对齐你问的 “Drift”）：

- Step31 中 `gps_status='Drift'` 的定义：原始 GPS 在中国境内且 `gps_dist_to_bs_m > 1500m`（阈值在 `lac_enbid_project/Layer_3/sql/31_step31_cell_gps_fixed.sql` 的 `params.drift_if_dist_m_gt`）。
- 本报告的距离：
  - `dist_to_old_center_m`：Step31 的 `gps_dist_to_bs_m`（原始点到 Step30 站中心）。
  - `dist_to_new_center_m`：将 `dist_to_old_center_m > 50km` 的点剔除后，用剩余点的 `AVG(lon_raw/lat_raw)` 生成“新中心”，再计算到新中心的距离（用于半径分布）。

## 总结：三者是否同类？

- `5G|1390240|2097247`：强“多质心 + 时间重叠” → 高概率碰撞/混桶（与 `cell_问题分析.md` 同类）。
- `5G|1389231|2097174`：主体在 2–10km，但被极端点拉爆（773km/402km）→ 更像“极端离群点混入/污染”，不是同款多质心。
- `5G|1394247|2097237`：主体 1km 内紧凑 + `STEP05_MULTI_LAC_CELL` → 结构性风险桶，非多质心。
