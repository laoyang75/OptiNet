# Phase1 快速启动与日常使用

更新时间：`2026-02-26`

## 1. 一键入口（根目录）
项目根目录提供 `launcher.py`，用于：
1. 启动服务
2. 关闭服务
3. 打开页面

## 2. 默认数据库配置
已默认指向当前数据库：
- Host: `192.168.200.217`
- Port: `5432`
- DB: `ip_loc2`
- User: `postgres`

默认环境在：
- `scripts/phase1_env.sh`

如需临时覆盖，可先导出同名环境变量，例如：

```bash
export PHASE1_DB_DSN='postgresql://user:pass@host:5432/dbname'
```

## 3. 常用命令
在仓库根目录执行：

```bash
python3 launcher.py start
python3 launcher.py status
python3 launcher.py open dashboard
python3 launcher.py open issues
python3 launcher.py open patches
python3 launcher.py stop
```

## 3.1 可点击控制台（推荐）
直接启动本地控制台（可点击启动/关闭/重启、看状态、打开页面）：

```bash
python3 launcher.py
```

默认地址：`http://127.0.0.1:8510/`

可选参数：
```bash
python3 launcher.py console
python3 launcher.py console --port 8512
python3 launcher.py console --no-open
```

重启并打开首页：

```bash
python3 launcher.py restart --open dashboard
```

## 4. 观测层构建与检查
构建 obs 数据：

```bash
scripts/run_phase1_obs_pipeline.sh
```

一致性检查：

```bash
scripts/check_phase1_obs_consistency.sh
scripts/check_phase1_obs_consistency.sh <run_id>
```

## 5. 测试回归
```bash
python3 tests/phase1_api_smoke_test.py
python3 tests/phase1_api_contract_test.py
```

## 6. 常见问题
1. `health` 显示 DB 未激活：
- 先执行 `python3 launcher.py env` 检查 DSN；
- 再执行 `python3 launcher.py restart`。
2. 页面打开但无数据：
- 先看 `python3 launcher.py status`；
- 若 API 不可达，页面会退回本地快照。
