# Phase1 可视化研究文件包

本目录用于支撑“分层可视化研究 + Agent协作修复”的完整闭环。

## 文件清单
- `01_需求理解_与_成功标准.md`：需求还原、成功标准、边界
- `02_分层可视化研究框架.md`：Layer_0~Layer_5 统一研究框架
- `03_可视化页面与数据模型蓝图.md`：页面结构、组件、接口字段
- `templates/Layer_级研究记录模板.md`：每层研究记录模板
- `templates/问题_补丁_决策日志模板.md`：问题发现到修复的闭环模板
- `templates/实验与操作日志模板.md`：Agent协作操作记录模板
- `prompts/phase1_可视化研究_深度Agent提示词.md`：给深度思考Agent的主Prompt

## 推荐使用顺序
1. 先读 `01` 与 `02`，统一目标与方法。
2. 再用 `03` 明确要做成什么页面与数据结构。
3. 执行研究时使用 `templates/` 三个模板持续记录。
4. 最后把主Prompt交给深度思考Agent执行。
