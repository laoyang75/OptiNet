1. 生命周期状态徽标

lifecycle_state -- 对象从注册到退出的全生命周期状态表达

等待 观察 活跃 休眠 退役 拒收
状态区分说明
等待 vs 观察：等待仅有圆点，观察带迷你进度条暗示"接近门槛"，且黄色略深
观察 vs 问题：观察是黄色 + 时钟（正在积累），问题是橙色 + 警告（有异常待处理）
问题 vs 拒收：问题是橙色（仍可流转），拒收是红色 + 删除线（已终结）
