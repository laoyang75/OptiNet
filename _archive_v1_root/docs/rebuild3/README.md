# rebuild3 最终文档包

本目录包含以下文件：

1. `00_审核结论.md`  
   两套文档的差异、冲突点与最终收敛决策。

2. `01_rebuild3_说明_最终冻结版.md`  
   rebuild3 的业务说明、状态模型、事实分层、版本与验证路径。

3. `02_rebuild3_预实施任务书_最终冻结版.md`  
   当前阶段（UI 设计前置）的任务边界、页面读模型、后续实施输入。

4. `03_rebuild3_技术栈要求_最终冻结版.md`  
   技术约束、schema 边界、实现原则与测试要求。

5. `04_agent_prompts_UI与开发.md`  
   两个独立 prompt：
   - Prompt A：UI 设计
   - Prompt B：UI 确认后生成实施计划并完成开发

建议阅读顺序：

`00_审核结论` → `01_说明` → `02_预实施任务书` → `03_技术栈要求` → `04_agent_prompts`
