## 角色分工

### Agent A：流程与数据契约审计
- 核查每层职责、输入输出、主键与分桶键、before/after/source/reason 字段。
- 对齐文档与 SQL，找口径漂移。

### Agent B：数据质量与一致性审计
- 核查对象存在性、行数链路、关键分布、异常标签闭环、指标表与事实表一致性。
- 量化问题规模和影响。

### Agent C：可视化与交接可读性方案
- 设计工程交接页面：总览、分层、契约、异常、验收。
- 输出前端可实现的组件和数据接口结构。

### Agent D：实时对接准备评估
- 识别现有批处理依赖和实时化阻塞点。
- 给出分阶段迁移路径和回滚策略。
