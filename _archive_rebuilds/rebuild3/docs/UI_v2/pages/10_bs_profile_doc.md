# BS 画像页 设计文档

## 页面定位

BS 画像页是 rebuild3 治理工作台中面向「基站」（Base Station）的专用画像/档案页面。聚焦 BS 维度的空间精度、健康状态、资格判定，是异常排查和数据质量评估的核心页面。

## 状态模型（P0）

### 主状态字段

| 字段 | 类型 | 说明 |
|------|------|------|
| lifecycle_state | 主状态 | 活跃 / 休眠 / 退役 |
| health_state | 主状态 | 健康 / 数据不足 / 碰撞嫌疑 / 碰撞确认 / 动态 / 迁移嫌疑 / GPS偏差 |
| anchorable | 资格 | 可锚定 / 不可锚定 |
| baseline_eligible | 资格 | 基线合格 / 基线不合格 |

### 参考信息（降级为解释层）

| 字段 | 原名 | 说明 |
|------|------|------|
| classification_v2 | 异常分类 → 旧分类（参考） | rebuild2 遗留分类，灰色文字展示 |
| gps_confidence | GPS 可信度 → GPS 可信度（参考） | rebuild2 遗留字段，灰色文字展示 |
| signal_confidence | 信号可信度 → 信号可信度（参考） | rebuild2 遗留字段，灰色文字展示 |

**重要：** 旧分类（classification_v2）、GPS 可信度、信号可信度三列降级为次要列，表头和数据均以灰色展示。展开详情中明确标注"rebuild2 参考信息"。

## 页面结构

### 1. 搜索栏

- 支持按 BS ID 或 LAC 搜索
- 精确匹配和前缀匹配

### 2. 筛选器

四个筛选维度：

| 筛选项 | 可选值 | 说明 |
|--------|--------|------|
| 运营商 | 全部/中国移动/中国联通/中国电信 | 按运营商过滤 |
| 制式 | 全部/4G/5G | 按网络制式过滤 |
| LAC | 全部/具体 LAC 编号 | 按所属 LAC 过滤 |
| 健康状态 | 全部/健康/数据不足/碰撞嫌疑/碰撞确认/动态/迁移嫌疑/GPS偏差 | 按健康状态过滤 |
| 资格 | 全部/可锚定/不可锚定/基线合格/基线不合格 | 按资格过滤 |

### 3. 汇总卡片

| 卡片 | 指标 |
|------|------|
| BS 总数 | 总数 + 活跃/休眠/退役分布 |
| 健康状态分布 | 健康/数据不足/碰撞嫌疑/碰撞确认/动态/GPS偏差 |

### 4. 主表格

17 列，支持点击表头排序：

| 列名 | 说明 | 列性质 |
|------|------|--------|
| 运营商 | 中国移动/联通/电信 | 属性 |
| 制式 | 4G/5G | 属性 |
| LAC | 所属 LAC 编号 | 属性 |
| BS ID | 基站标识 | 属性 |
| 生命周期 | 活跃/休眠/退役 | **主状态** |
| 健康状态 | 健康/数据不足/碰撞嫌疑/碰撞确认/动态/迁移嫌疑/GPS偏差 | **主状态** |
| 资格 | 可锚定/不可锚定 + 基线合格/基线不合格 | **主状态** |
| Cell 数 | 下辖 Cell 数量 | 统计 |
| P50(m) | GPS 点到质心的 P50 距离 | 质量指标 |
| P90(m) | GPS 点到质心的 P90 距离 | 质量指标 |
| 面积 km² | 凸包面积 | 质量指标 |
| GPS 原始率 | 有原始 GPS 的记录占比 | 质量指标 |
| 信号原始率 | 有原始信号值的记录占比 | 质量指标 |
| RSRP | RSRP 均值 (dBm) | 质量指标 |
| 旧分类（参考） | 正常/动态BS/碰撞确认/碰撞嫌疑/面积大/GPS噪声 | **参考信息，灰色** |
| GPS 可信度（参考） | 高/中/低/无 | **参考信息，灰色** |
| 信号可信度（参考） | 高/中/低/无 | **参考信息，灰色** |

### 5. 行展开详情

点击行展开四个区块：

#### 5.1 空间精度
中心点坐标、P50/P90/最大距离、面积。

#### 5.2 对象状态
- lifecycle_state、health_state、anchorable、baseline_eligible
- 分隔线下方：rebuild2 参考信息（旧分类、GPS 可信度、判定依据等，灰色文字）

#### 5.3 GPS 来源构成
堆叠条图：原始 / Cell 补齐 / BS 补齐 / BS 风险补齐。

#### 5.4 信号质量
RSRP/RSRQ/SINR 均值。

### 6. 口径说明

每个详情面板底部附带口径说明，解释当前 BS 的状态判定逻辑。

### 7. 分页 + 排序

- 底部分页控件，每页 10 条
- 表头可点击排序

## 数据依赖

- `obj_bs` 表：BS 注册信息、lifecycle_state、health_state、anchorable、baseline_eligible、classification_v2（旧）、gps_confidence（旧）
- `obj_cell` 表：聚合统计 Cell 数量
- `fact_governed` / `fact_pending_*` 表：聚合统计记录数、GPS/信号来源构成
- `baseline_bs` 表：质心坐标、P50/P90 距离
