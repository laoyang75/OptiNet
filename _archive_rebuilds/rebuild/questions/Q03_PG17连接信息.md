# Q03: PG17 连接信息

## 问题

PG17 数据库的连接信息是什么？

## 背景

当前已知的连接信息：
- PG15：`postgresql://postgres:123456@192.168.200.217:5432/ip_loc2`（已验证可连接）

你提到数据已从原始数据库导入了 PG17，但我无法连接到 PG17。尝试过的地址：
- `localhost:5432` → 连接被拒绝
- `localhost:5417` → 连接被拒绝
- MCP 工具中未找到 PostgreSQL 连接器

## 需要你提供

请提供以下信息：
- PG17 主机地址（IP 或 hostname）
- 端口号
- 数据库名
- 用户名
- 密码
- 数据是否已完整导入（56 张 Y_codex 表是否都在）

## 影响

PG17 连接信息决定了：
1. 后续所有 DDL 和 SQL 的执行目标
2. 能否直接在 PG17 上验证表结构
3. session_restore prompt 中的连接配置
