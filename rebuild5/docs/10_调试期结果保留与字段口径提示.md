## 2. 当前最容易混淆的字段口径

### 2.1 snapshot 与 library 不是同一个东西

- `trusted_snapshot_*`：Step 3 当前批冻结视图
- `trusted_*_library`：Step 5 维护后正式发布结果

UI 和脚本都不能把它们混成同一层。

### 2.2 Cell 碰撞有两套语义

- `collision_id_list`：B 类 `cell_id` 防护表，给 Step 2 / Step 4 用
- `trusted_cell_library.is_collision`：A 类真碰撞治理标签

两者不能互相替代。

### 2.3 以代码实际字段为准，不用文档旧别名推断

当前若要确认具体字段，请先看：

- `rebuild5/backend/app/maintenance/schema.py`
- `rebuild5/backend/app/enrichment/schema.py`
- 当前库中对应表的真实 schema

设计文档是语义基线，具体字段名和是否已落地，以代码和数据库为准。
