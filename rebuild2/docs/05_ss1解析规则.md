# SS1 解析规则（确认版）

> 确认日期：2026-03-31
> 状态：**已确认**，替代之前 03_字段治理规则.md 中的 ss1 部分

---

## 1. SS1 完整格式

```
{信号块}+&{获取时间}&{位置信息}&{基站信息}+&{AP信息};{下一组}...
```

### 示例

```
l,-78,-78,-14,12+&1764806695&116.519630,40.021122,1764806654212&n,539066369,73775,46011+l,156385938,6326,46011+&0
```

---

## 2. 五个元素（按 `&` 分割）

| # | 名称 | 示例 | 说明 |
|---|------|------|------|
| ① | 信号强度 | `l,-78,-78,-14,12+` | 按 `+` 分多条，每条 5 值：制式,SS,RSRP,RSRQ,RSSNR |
| ② | 获取时间 | `1764806695` | Unix 秒级时间戳 |
| ③ | 位置信息 | `116.519630,40.021122,1764806654212` | 经度,纬度,定位时间戳（毫秒） |
| ④ | 基站信息 | `n,539066369,73775,46011+l,156385938,6326,46011+` | 按 `+` 分多条（双卡），每条 4 值：制式,CellID,LAC,PLMN |
| ⑤ | AP信息 | `0` 或 `bssid,ssid` | WiFi 信息，当前阶段不提取 |

---

## 3. 信号块详细格式

```
{tech},{SignalStrength},{RSRP},{RSRQ},{RSSNR}
```

| 位置 | 字段 | 映射到 L0 | 说明 |
|------|------|----------|------|
| 1 | tech | — | `l`=LTE, `n`=NR, `g`=GSM（用于匹配基站） |
| 2 | SignalStrength | sig_ss | 通用信号强度 |
| 3 | RSRP | sig_rsrp | 参考信号接收功率 |
| 4 | RSRQ | sig_rsrq | 参考信号接收质量 |
| 5 | RSSNR | sig_sinr | 信号与干扰噪声比 |

**有效性规则**：RSRP = -1 或 RSRQ = -1 视为无效，置 NULL

---

## 4. 基站块详细格式

```
{tech},{CellID},{LAC},{PLMN}
```

| 位置 | 字段 | 映射到 L0 | 说明 |
|------|------|----------|------|
| 1 | tech | tech_raw/tech_norm | `l`=LTE→4G, `n`=NR→5G, `g`=GSM→2G |
| 2 | CellID | cell_id_dec | Cell ID，-1 为无效 |
| 3 | LAC | lac_dec | LAC/TAC |
| 4 | PLMN | operator_id_raw | 运营商编码 |

**有效性规则**：CellID = -1 视为无效，不产出行

---

## 5. 核心匹配规则：信号按制式绑定基站

**一组 ss1 数据中，信号和基站按制式类型匹配：**

```
信号 l,-78,-78,-14,12  →  绑定到基站 l,156385938,6326,46011
信号 n,...             →  绑定到基站 n,539066369,73775,46011
```

**每对匹配产出一行 L0 数据。** 如果某个基站没有对应制式的信号，该行信号字段为 NULL。

### 双卡场景

手机有两张卡（如联通 NR + 电信 LTE），基站块有两条：
```
n,539066369,73775,46011+l,156385938,6326,46011+
```
信号块可能只有一条（当前活跃卡的信号），也可能有两条。

---

## 6. 继承规则

| cell_block 值 | 含义 | 处理 |
|--------------|------|------|
| `l,xxx,xxx,xxx+...` | 有效基站 | 正常解析，产出行 |
| `1` | 继承上一组 | 沿用前一个有效组的基站块 |
| `0` 或空 | 无基站 | 跳过，不产出行 |
| `g,-1,-1,plmn` | 只有运营商 | CellID=-1，不产出行 |
| `c,-1,-1,plmn` | CDMA 无效 | 同上 |

---

## 7. 特殊值处理

| 值 | 含义 | 处理 |
|----|------|------|
| CellID = -1 | 无效 | 不产出该行 |
| RSRP = -1 | 未获取 | 置 NULL |
| RSRQ = -1 | 未获取 | 置 NULL |
| RSSNR = -1 | 未获取 | 置 NULL |
| SS = -1 | 未获取 | 置 NULL |
| GPS = `0` | 无定位 | 经纬度置 NULL |
